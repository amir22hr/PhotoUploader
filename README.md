
# Photo Uploader

  

**Node - Express - Bulma - MySQl**

  

1. Start **MySQL**

2. Install Package in Project

  

> npm install

  

3. Config **variables.env**

4. RUN **npm start**


## DATABASE Structure


- DataBase Name: **PhotoUploader**

- Table Name: **images**

- id - AUTO_INCREMENT - varchar 11

- user_ip - varchar 15

- name - varchar 255

- format - varchar 4

- size - varchar 11

- date - Date
